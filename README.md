## Getting Started

1- Clone the project
```
git clone https://github.com/FMGordillo/WomenWorkshop2017.git
```
2- Install all the dependencies
```
npm install
```
3- Run the server (we recommend to use nodemon)
```
nodemon
```
4- Go on and check it out
```
http://localhost:3000
```
